# FC-Midtjylland---UEFA-Europa-League-MD-6-8-
A tactical analysis of the last 3 League Phase matches for FC Midtjylland in the 2025-26 UEFA Europa League. 
Focused on generating a passing netowrk visualization from the mentioned fixtures, along with with shot maps indicating the xG value of each attempt, and finally, xG timelines. 
This was done as part of my Master´s in Big Data and Data Analysis in Football at Real Madrid Graduate School.
